## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.

## Usage
- For Backend
1. Start the server using `npm start`.
2. Access the application at [http://localhost:3000](http://localhost:3000).
- Fro Frontend
```terminal
cd frontend
Index.html start with Live server
```
